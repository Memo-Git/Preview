# GitHub Pages 部署说明

本目录是星火医疗 App 原型的 GitHub Pages 静态站点目录。

## GitHub Pages 设置

1. 将仓库推送到 GitHub。
2. 打开仓库 Settings -> Pages。
3. Source 选择 Deploy from a branch。
4. Branch 选择 main，Folder 选择 /docs。
5. 保存后等待 GitHub Pages 构建完成。

访问地址通常为：

```text
https://<github用户名>.github.io/<仓库名>/
```

## 说明

- 不需要 Node.js。
- 不需要 Python。
- GitHub Pages 会直接托管本目录内的 HTML、CSS、JS、图片和图标。
- 当前内容是移动端 App 原型网页预览，建议用手机浏览器或桌面浏览器的移动端画布查看。
